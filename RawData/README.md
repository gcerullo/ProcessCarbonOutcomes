# RawData (redistributable)

This folder contains the **raw plot-level inputs** used to fit the carbon recovery model in the Nature Revision 2 pipeline.

## Required files

- `SSB_plot_data.csv`
  - Used by `Scripts/Nature_Revision_2/01_one_model_to_rule_them_all.R`
  - Plantation inventory measurements used to derive plantation ACD through time.

- `Philipson20_PlotData2.csv`
  - Used by `Scripts/Nature_Revision_2/01_one_model_to_rule_them_all.R`
  - Primary / logged / restored plot data (from Philipson et al.) used to fit recovery trajectories.
